# add-tests Skill

## Purpose

Add unit, integration, or E2E tests for existing or new code.

## When to Use

- "Add tests for X"
- "Write E2E test for flow Y"
- "Add coverage for Z"

## Steps

1. Identify the test type (unit, integration, E2E).
2. Choose framework (Jest, Vitest, Playwright, etc.).
3. Write tests covering happy path and edge cases.
4. Run tests and ensure they pass.
