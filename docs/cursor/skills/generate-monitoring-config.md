# generate-monitoring-config Skill

## Purpose

Generate Datadog or Cloudflare monitoring config (dashboards, alerts).

## When to Use

- "Set up monitoring for service X"
- "Create a dashboard for..."
- "Add alerts for error rate"

## Steps

1. Identify metrics to track (latency, errors, throughput).
2. Use Datadog or Cloudflare observability MCP.
3. Create dashboard with relevant widgets.
4. Configure alerts with thresholds and notification channels.
