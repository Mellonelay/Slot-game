# reverse-engineer-flow Skill

## Purpose

Extract flows, APIs, and protocols from external systems for implementation.

## When to Use

- "Reverse engineer the flow for URL X"
- "Extract the API from..."
- "Scrape and document how Y works"

## Steps

1. Use Firecrawl to crawl/scrape the target.
2. Use Bright Data or cursor-ide-browser for interactive flows.
3. Document endpoints, payloads, auth flow.
4. Produce structured findings (JSON, markdown) for Backend/Frontend.
